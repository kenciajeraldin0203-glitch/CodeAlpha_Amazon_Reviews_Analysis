# Amazon Reviews Analysis

## Project Overview

This project analyzes Amazon product reviews using Python. The objective is to perform Exploratory Data Analysis (EDA), Data Visualization, and Sentiment Analysis to understand customer opinions and product performance.

## Tools Used

* Python
* Pandas
* Matplotlib
* TextBlob
* Jupyter Notebook

## Tasks Completed

### Exploratory Data Analysis (EDA)

* Explored dataset structure and data types
* Identified missing values
* Analyzed ratings and recommendations
* Examined brand distribution

### Data Visualization

* Rating Distribution Chart
* Sentiment Distribution Chart

### Sentiment Analysis

* Classified reviews as Positive, Negative, and Neutral
* Generated sentiment scores using TextBlob

## Key Findings

* Dataset contains 1597 records and 27 columns
* Average Rating: 4.36/5
* Positive Reviews: 1479
* Negative Reviews: 77
* Neutral Reviews: 41

## Conclusion

The analysis indicates that customers are generally satisfied with the products. Most reviews are positive, and the overall rating trend reflects strong customer satisfaction.
